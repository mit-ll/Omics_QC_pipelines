## Overview
```
Folders:
qc_base			QA/QC base for Singularity pipelines
qc_atac_seq		ATAC-seq pipeline
qc_chipmentation	Chipmentation pipelin
qc_epic			EPIC Pipeline
qc_mass_cytof		Mass CyTOF pipeline
qc_medip_seq		Methylated DNA Immunoprecipitation Sequencing (MeDIP-seq) pipeline
qc_mint_chip		MINT-ChIP-seq pipeline
qc_mirna		Micro RNA-seq (miRNA-seq) pipeline (smaller datasets)
qc_mirna_blast		Micro RNA-seq (miRNA-seq) pipeline
qc_multiome		Multiome pipeline
qc_multiome_ranger	Multiome pipeline with 10X CellRanger-arc results
qc_rna_seq		RNA-seq pipeline
qc_sc_atac_seq		Single cell ATAC-seq pipeline
qc_sc_atac_seq_ranger	Single cell ATAC-seq pipeline with 10X CellRanger (ATAC-seq) results
qc_sc_rna_seq		Single cell RNA-seq pipeline
qc_sc_rna_seq_ranger	Single cell RNA-seq pipeline with 10X CellRanger (RNA-seq) results
qc_snmC_seq		Single cell DNA methylome profiling - snmC-seq/snmC2-seq pipeline
qc_wgbs			Whole Genome Bisulfite Sequencing (WGBS) pipeline
```
